Ajuda d'Estadístiques Globals

Mostrar Estadístiques Globals

Aquest apartat ens ajuda a mostrar les estadístiques globals d'un algorisme seleccionat.

Passos a seguir:

  1. Seleccionar un tipus de descompressió dins del requadre anomenat Algorisme.
  2. Apareixen automàticament les Estadístiques Globals de l'algorisme seleccionat.

Exportar Estadístiques Globals

Aquest apartat ens ajuda a exportar les estadístiques globals d'un algorisme seleccionat.

Passos a seguir:

  1. Seleccionar un tipus de descompressió dins del requadre anomenat Algorisme.
  2. Apareixen automàticament les Estadístiques Globals de l'algorisme seleccionat.
  3. Prèmer el botó anomenat "Exportar".
  4. Apareix una nova pantalla on pots seleccionar on guardar el fitxer .txt.
  5. Per visualitzar-les, buscar on s'ha guardat el fitxer amb el nom "EstGlobals.txt".

Per informació més detallada consulta el Manual d'Usuari inclòs al projecte.
